# Решение соревнования Titanic на Kaggle (Top 77%)

В данном репозитории представлено мое решение классического соревнования на Kaggle по предсказанию выживаемости пассажиров Титаника.

## Основные этапы работы:

* **Исследовательский анализ данных (EDA)** и визуализация зависимостей через `matplotlib` и `seaborn`.
* **Feature Engineering:** извлечение титулов пассажиров (`Title`), умное заполнение пропусков.
* **Предобработка:** кодирование категориальных признаков через One-Hot Encoding и стандартизация данных (`StandardScaler`).
* **Моделирование:** обучение логистической регрессии с валидацией (`stratify`).

**Результат на Kaggle:** Score = 0.77033
